Detailed Description

Comprehensive portfolio performance metrics.

This structure contains all standard and advanced performance metrics used in institutional portfolio analysis. All values are calculated using industry-standard methodologies and are suitable for regulatory reporting and risk management.

Note
All ratio-based metrics (Sharpe, Sortino, etc.) are annualized
VaR metrics are expressed as positive values (loss magnitude)
Beta and Alpha are calculated relative to the provided benchmark
