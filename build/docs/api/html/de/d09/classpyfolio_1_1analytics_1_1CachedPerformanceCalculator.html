Detailed Description

High-performance cached performance metrics calculator.

This class provides intelligent caching for expensive performance metric calculations. It uses content-based hashing to detect data changes and automatic cache invalidation based on time and data changes.

Features:

  • Content-aware caching (detects data changes via hashing)
  • Automatic cache invalidation and cleanup
  • Thread-safe operations with read-write locks
  • Performance monitoring and adaptive caching
  • Memory-efficient with configurable limits
  • Hit rate optimization
