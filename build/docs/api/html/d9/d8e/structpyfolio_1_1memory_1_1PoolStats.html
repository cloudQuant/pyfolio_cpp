Detailed Description

High-performance memory pool allocator for financial data structures.

This allocator is designed for high-frequency trading scenarios where allocation speed is critical and memory fragmentation must be minimized.

Features:

  • O(1) allocation and deallocation
  • Thread-safe operations
  • Memory alignment support
  • Configurable pool sizes
  • Memory usage tracking
  • Automatic growth when needed
