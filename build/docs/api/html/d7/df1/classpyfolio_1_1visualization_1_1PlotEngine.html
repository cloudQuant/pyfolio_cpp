Detailed Description

Simple plotting interface that can generate multiple output formats.

This class provides plotting functionality without requiring matplotlib. It can generate:

  • HTML plots using embedded JavaScript (plotly.js/d3.js)
  • SVG plots
  • CSV data for external plotting
  • Simple ASCII plots for console output
